## 算法笔记示例

每篇笔记包含：
- 算法原理与核心思想
- 代码模板（可直接复用）
- 复杂度分析
- 常见易错点

### 示例：深度优先搜索 DFS

```cpp
void dfs(int u) {
    vis[u] = true;
    for (int v : adj[u]) {
        if (!vis[v]) {
            dfs(v);
        }
    }
}
```
